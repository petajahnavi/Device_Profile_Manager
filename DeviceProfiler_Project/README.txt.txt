Project Name: DeviceProfiler

Description:
This project profiles smartphone compute capabilities using lightweight techniques.

Modules:
1. Android App – collects device info, runs benchmark, calculates score
2. Backend – Node.js server with MongoDB to store device profiles

How to Run:

Android:
- Open in Android Studio
- Run on device/emulator

Backend:
- Open backend folder in VS Code
- Run: npm install
- Run: node server.js

API:
POST /profile
GET /profile
